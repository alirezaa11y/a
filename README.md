# 📚 دستورات ربات :

#🔹 سکوت [ریپلای]
# - اضافه کردن کاربر به لیست سکوت (فقط ادمین)

🔹 حذف سکوت [ریپلای]
 - حذف کاربر از لیست سکوت (فقط ادمین)

🔹 لیست سکوت
 - نمایش لیست کاربران ساکت شده (فقط ادمین)

🔹 پاکسازی لیست سکوت
 - پاک کردن تمام لیست سکوت (فقط ادمین)

🔹 پین [ریپلای]
 - سنجاق کردن پیام (فقط ادمین)

🔹 بن [ریپلای]
 - بن کردن کاربر (فقط ادمین)
